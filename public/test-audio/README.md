# test-audio — 3단계 재생 엔진 테스트 음원

3단계(재생 엔진) 검증용 트랙 세트. 모두 **PCM 44100Hz / 스테레오 / 16-bit / 19.2초**
(동일 길이·BPM·포맷).

| 파일 | 용도 |
|------|------|
| `drums.wav` | 드럼 트랙 — `drumGain`(드럼 볼륨 슬라이더)으로 0~100% 조절 |
| `backing.wav` | 반주 트랙(드럼 없음) — `backingGain` 항상 1.0 |
| `mix-reference.wav` | 합본 참고용 — 두 트랙을 같이 재생하면 이 소리가 기준. 앱은 로드하지 않음(개발/청취 대조용) |

재생 엔진(`src/lib/audio-engine.ts`)은 "두 AudioBuffer를 받아 동시 재생"하는
에셋 비의존 구조다. 4단계에서 입력이 이 test-audio → 실제 곡의 분리 결과로
교체되며, 그 외 엔진 코드는 그대로 쓰인다.
